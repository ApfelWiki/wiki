A) Setting up stylized - top

stylized is an HTML theme for your business / portoflio. It's a perfect way for you to show your services and give the users some examples of your work in a very stylish way. This theme uses a fixed layout. All of the information within the main content area is nested within a div with an id or class.

To install the theme, put the stylized-Theme folder in the root directory of your website (you don't have to upload the 'Help' and 'PSD' map)

Setting it Up.
Note: you can use any text-editor you like to edit the files.

Step 1 - The Home page

The home page uses a slider and has 3 columns to explain your services. To change the slider images you can do the following:

  • Create you own images and make sure is has a width of 819px and a height of 312px
  • Go to the 'images' map and upload your images into that map.
  • Open up the index.html with a text-editor and edit the paths to the images on line 83.

You can change the text in the columns the same way:

  • Open up the index.html again and search for the 'INFO BOX' div (line 97).
  • Copy in your own text and save the file


Step 2 - The Portfolio page

The portfolio page is where you can showcase your work. There's a slider at the top of the page with thumnails. When a visitor clicks the thumbnail, a project will be shown underneath it. To add your own thumbnails, showcase preview images and project info you can do the following:

  • Create you own thumbnails and make sure they have a width of 209px and a height of 108px
  • Create you own showcase images and make sure they have a width of 502px (the height isn't important.
  • Go to the 'images/projects' map and upload your images into that map.
  • Open up the portfolio.html with a text-editor and edit the paths to the images (see following screenshot).
  • To change the text, open up the portfolio.html again and replace the text in the particular 'item' division.

Currently there are 9 examples of portfolio items. You can change this yourself by deleting or adding the lines of codes that belong to the portfolio item.


Step 3 - The Services page

The services page is where you want to explain to your visitors what it is that you do. To edit this page do the following:

  • Open up services.html with your text-editor.
  • Replace the text in the 'Services box' that is already there with your own text.
  • Make sure that you change the h2-element with your page title, the h3-element with your service title and the p-element with the information on the service.

To change the testimonials on the right side of the page you do the following:

  • Open up services.html with your text-editor.
  • Replace the text that is already there with your own text.


Step 4 - The Contact page

The contact page is has a contact form on the left side and an image with address info underneath it on the right side. The form itself is just based on elements and does not work without any PHP code. You'll need to contact a scripter to build you a script for this form or, if you know what you're doing, do it yourself with one of the many contact form scripts out there.

B) CSS Files and Structure - top

I'm using four CSS files in this theme. Two of these files are for the sIFR type in the pages, you don't need to worry about these. The other two are responsible for the global look of the template.

  1. global.css
  2. ie6.css
  3. sifr.css
  4. sifr-font.css

The main global.css file contains all of the specific stylings for the pages. The file is separated into sections using:

	
/* ### GENERAL ### */

	some code

/* ### HEADER CONTAINER ### */
	
	some code
	
/* ### MENU CONTAINER ### */

	some code
	
/* ### CONTENT CONTAINER ### */

	some code
	
/* ### FOOTER CONTAINER ### */

	some code
	
	etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports 6 Javascript files.

  1. jQuery
  2. sIFR
  3. Sliders

jQuery is a Javascript library that greatly reduces the amount of code that you must write. The animation in this site is carried out from the slider scripts. There's no need to change anything in here except for the animation time of the featured slider.

  • To change the time loop of the slider on the homepage, just open up 'javascripts/slider.js' and change the 800 on line 51 to any number you like (1000 = 1 second).
  • To change the time loop of the slider on the portfolio page, just open up 'javascripts/sslider.js' and change the 100 on line 53 to any number you like (1000 = 1 second).
